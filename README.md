
## Local Development

### Prerequisites

* Node.js (v18 or higher recommended)
* pnpm

---

### Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/supernano-bot.git
cd supernano-bot
pnpm install
```

---

### Development

Run the CLI in development mode:

```bash
pnpm dev
```

---

### Build

Compile TypeScript to JavaScript:

```bash
pnpm build
```

---

### Run Locally

After building:

```bash
pnpm link --global
supernano-bot
```
